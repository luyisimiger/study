# Docker

## Docker images

Una **imagen docker** es un parquete de lectura, tiene todo lo necesario para ejecutarse. _Un paquete todo en uno._

> Una **imagen** docker es como una **_clase_** y cada **contenedor** es como un **_objeto_**

### Docker registry

Lugar donde se pueden desplegar nuestras imagenes de docker

```docker
docker pull [image-name]
```
