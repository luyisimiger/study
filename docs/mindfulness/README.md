
# Mindfulness
> Traducción literal: Atención plena

## Que es

- Estar en el presente de manera consciente
- Es una técnica para enfocarse en el "aquí y ahora"

## Para que sirve

- Cuando vives en el pasado, entras en **depreción**
- Cuando vives en el fututo, entras en ansiedad
